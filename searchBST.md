# Search BST

Add a method called `search` on the `BinaryTree` that receives un number and returns true if the number is on the tree, false otherwise.

```javascript
const tree = new BinaryTree();
tree.add(4);
tree.add(2);
tree.add(7);
tree.add(1);
tree.add(3);

tree.search(7); // true
tree.search(10); // false
```
